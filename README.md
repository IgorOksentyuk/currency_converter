**Application in which you can view the list of products and add/remove them to the cart. Also you can open admin panel in which you can add new products and remove current.**

---

_Made with plain:_

- HTML;
- SCSS (BEM);
- TS;
- Angular;
- Angular Material;
- RxJS.

---

_How to run:_

1. npm install
2. npm install -g @angular/cli
3. ng serve
